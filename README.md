# java 反射实战教程

Java反射，本质上是Java程序能够对自身程序要素（类名、方法、注解等）进行自我检查及自我修改的能力，是通过维护语言层级元信息的手段来有效地对应用层级进行解耦的解决方案。因而，它必须由Java虚拟机从底层提供支持。
